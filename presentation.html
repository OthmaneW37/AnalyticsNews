Architecture Medaillon

Le Data Lake suit le modele Medaillon en 3 couches progressives : chaque couche affine la qualite et la valeur des donnees.

Couche 1

🟤 Bronze — Donnees brutes

Donnees telles qu'elles sortent des scrapers, sans transformation.

  • Format JSON partitionne par source/date
  • Stockage local ou MinIO (s3://bronze)
  • Champs : titre, url, source, langue, date, contenu, pays
  • Aucun controle qualite
  • Retention : 90 jours
Couche 2

⚪ Silver — Donnees nettoyees

Nettoyees, normalisees et validees, pretes pour l'analyse.

  • Format Parquet + JSON
  • Suppression HTML, URLs, caracteres de controle
  • Normalisation des champs
  • Detection automatique de la langue
  • Deduplication par article_id
  • Controle qualite 3 dimensions
Couche 3

🟡 Gold — Donnees enrichies

Agregees et enrichies avec IA et signaux de marche.

  • Topic modeling BERTopic (multilingue)
  • Signaux Polymarket (marches predictifs)
  • Score de couverture par topic
  • Signal combine (probabilite x pertinence)
  • Tables analytiques : jour, theme, pays, source
  • Alimentation du Data Warehouse DuckDB
🔄

Pipeline de Donnees

📦 Mode Batch

  • 🕐 Scraping programme toutes les heures
  • 🛠️ Orchestre par Apache Airflow (DAGs)
  • 📊 Traitement complet Bronze → Silver → Gold
  • 🗄️ Alimentation du Data Warehouse
  • 📋 CLI : run_full_pipeline.py
  • ⚡ Fallback sans Docker possible

⚡ Mode Streaming

  • 📡 Evenements Kafka en temps reel
  • 🔄 Consumer traite chaque article immediatement
  • 📦 Topic news_events
  • 📊 Ingestion Silver en continu
  • 🐳 Broker Kafka KRaft (sans ZooKeeper)
  • 🌐 Port 9092 expose

📋 Etapes ETL

Stockage

1. Scraping & Bronze

Collecte RSS/HTML → JSON partitionne par source/date.

Streaming

2. Evenement Kafka

Emision d'evenements vers le topic news_events.

Qualite

3. Silver — Nettoyage

Nettoyage HTML, langdetect, deduplication, controle qualite.

Enrichissement

4. Gold — BERTopic + Polymarket

Topic modeling IA + signaux marches predictifs.

Analyse

5. Data Warehouse

Insertion DuckDB + tables analytiques + vues SQL.

Visualisation

6. Dashboard Streamlit

KPIs, graphiques de topics, signaux, table interactive.
